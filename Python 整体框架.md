## 三、子模型 1：日本汽车在美国市场的基准地位

### 1.1 分组与指标设计

先把美国新车市场分成若干“来源类别”：

* **US_dom**：美国本土品牌 + 在美国生产的外资品牌（含日系在美生产车）
* **JP**：日本本土生产、直接出口到美国的整车
* **MX**：在墨西哥生产出口美国的整车（含日系品牌在墨西哥工厂）
* **CA**：加拿大生产出口美国的整车
* **KR / EU / Others**：韩国、欧盟及其他国家

你需要构造的**核心基准指标**：

* 美国新车总销量：(Q^{US}_{2024})（可用行业统计或题中参考文献 [5]）
* 日本品牌在美国销售量：(Q^{JP,brand}_{2024})（可以分解为在美生产 + 进口）
* 日本**汽车进口量**（从日本直接出口到美）：
  [
  Q^{import}*{JP,2024},\quad V^{import}*{JP,2024}
  ]
* 其他主要来源国进口量与金额：( Q^{import}*{r,2024}, V^{import}*{r,2024} )

从而得到：

* 日本车在美国新车市场的**销量占比**：
  [
  s^{brand}*{JP}=\frac{Q^{JP,brand}}{Q^{US}*{total}}
  ]
* 日本直接对美整车出口在美国**新车销售中的占比**：
  [
  s^{imp}*{JP}=\frac{Q^{import}*{JP}}{Q^{US}_{total}}
  ]
* 以及**美国汽车进口结构**（日本、墨西哥、加拿大、韩国、欧盟等各自占比）

参考资料显示，2024 年美国轻型车销量约 1600 万辆，其中约 46% 为进口车，进口来源国以墨西哥、韩国、日本和加拿大为主，这与你题目中的 46% 说法吻合。([S&P Global][1])

这些基准份额将用于**标定后续的需求/替代弹性参数**。

## 四、子模型 2：关税–价格–需求–进口结构（核心）

核心思想：

> “不同来源国的汽车是品质相近但来源不同的差异化产品，消费者在不同来源间有替代；关税改变的是各来源国在美终端售价，从而改变进口结构和总需求。”

### 2.1 变量和索引

* 年份：(t = 2024, 2025,\dots)
* 来源国 / 地区：(r \in {US_dom, JP, MX, CA, KR, EU, OTH})
* 基期（关税调整前）的**平均进口车价**（不含关税）：
  [
  P_{r,0}
  ]
* 基期进口量：
  [
  Q_{r,0}
  ]
* 基期市场份额：
  [
  s_{r,0} = \frac{Q_{r,0}}{\sum_k Q_{k,0}}
  ]
* 基期进口关税率（对整车）：(\tau_{r,0})（从关税数据库算）
* 调整后的新关税率：(\tau_{r,1})（包含“10% 基础 + 追加 11–50%”的互惠关税）

### 2.2 市场份额模型：多国 Armington / logit 形式

为了兼顾可算性与直观性，建议使用**多项 logit 形式**的市场份额模型：

设关税后消费者面对的**含税终端价格**：

[
P_{r} = P_{r,0} \cdot (1+\tau_{r})
]

令 (\sigma>0) 为来源国间的替代弹性参数，(A_r) 为偏好 / 质量参数，则**来源国 r 的市场份额**：

[
s_r(\boldsymbol{\tau}) = \frac{A_r , P_r^{-\sigma}}{\sum_k A_k , P_k^{-\sigma}}
]

在基期（(\boldsymbol{\tau} = \boldsymbol{\tau}*0)）我们观测到 (s*{r,0})、(P_{r,0})，可以用它们来**标定 (A_r)**：

[
A_r = s_{r,0},P_{r,0}^{\sigma}
]

因为此时分母 (\sum_k A_k P_k^{-\sigma} = \sum_k s_{k,0}=1)。

关税调整后，只要给出新关税 (\tau_{r,1}) 和假定的替代弹性 (\sigma)，就能算出新的份额：

[
s_{r,1} =
\frac{s_{r,0},P_{r,0}^{\sigma},\big(P_{r,0}(1+\tau_{r,1})\big)^{-\sigma}}
{\sum_k s_{k,0},P_{k,0}^{\sigma},\big(P_{k,0}(1+\tau_{k,1})\big)^{-\sigma}}
]

这个公式就可以回答：

* 日本对美整车出口份额 (s_{JP,1}) 如何变化
* 美国整体进口中，日本、墨西哥、加拿大等来源国的份额重新分配情况

### 2.3 总体需求对平均车价的反应

上面只是**份额**，还需要整体销量的变化。可用简单的**价格–需求弹性模型**：

设：

* 基期平均车价：
  [
  \bar P_0 = \sum_r s_{r,0} P_{r,0}(1+\tau_{r,0})
  ]
* 新均价：
  [
  \bar P_1 = \sum_r s_{r,1} P_{r,0}(1+\tau_{r,1})
  ]
* 新车总需求：
  [
  Q_{\text{total},1}=Q_{\text{total},0}\left(\frac{\bar P_1}{\bar P_0}\right)^{-\eta}
  ]
  其中 (\eta) 是**总需求的价格弹性**（可以参考行业文献，通常在 0.8–1.5 之间，或用历史数据回归估计）。

于是各来源国的销量为：

[
Q_{r,1}=s_{r,1},Q_{\text{total},1}
]

这样你就完成了**“关税 → 价格 → 总需求 + 进口结构 → 各国出口量”**这一条主线。

> 注意：如果你想简化，也可以假设总需求 (Q_{\text{total}}) 不变，只调整份额。那就把上面的价格需求弹性模块删掉即可。

## 五、子模型 3：日本的非关税应对策略（如何嵌入模型）

题目明确要求“纳入日本的非关税应对策略”，现实中主要包括：

* **生产基地迁移 / 扩大在美或墨西哥生产**（FDI，本地化）
* **内部消化部分关税**（降利润率、价格折扣 → 关税部分不完全向消费者转嫁）
* **调整产品结构**（高端/电动车占比变化）

这里给一个**既有经济含义又不太难算的刻画方式**。

### 3.1 “有效关税率” + 关税转嫁程度

对某个来源 r（例如“日本本土出口美国”），设名义关税为 (\tau^{nom}_r)，日本企业通过降价吸收关税的比例为 (\theta_r\in[0,1])，则消费者感受到的**有效关税率**：

[
\tau^{eff}_r = (1-\theta_r)\tau^{nom}_r
]

于是 2.2 中的价格改为：

[
P_r = P_{r,0},(1+\tau^{eff}*r)
= P*{r,0},\big(1+(1-\theta_r)\tau^{nom}_r\big)
]

不同情景可以设不同的 (\theta_{JP})：

* 情景 A：不吸收关税（(\theta_{JP}=0)）
* 情景 B：吸收 50%（(\theta_{JP}=0.5)）
* 情景 C：激烈价格战，(\theta_{JP}=0.8)

### 3.2 生产基地迁移：把“日系在美/墨生产”显式化

引入“日系品牌在美/墨生产份额”的变量：

* ( \lambda^{US}_{JP} )：日系品牌在美国本土生产占其在美销量的比例
* ( \lambda^{MX}_{JP} )：日系品牌在墨西哥生产占其在美销量的比例
* ( \lambda^{JP}_{JP} )：日本本土出口直接供美国市场的比例

基期满足：

[
\lambda^{US}*{JP,0} + \lambda^{MX}*{JP,0} + \lambda^{JP}_{JP,0} = 1
]

根据行业资料，丰田、本田等已经有很高比例的北美本地化生产，你可以用行业报告/新闻估一个基期值。([S&P Global][1])

**关税上调后的反应函数**可以设为线性或 S 形函数，例如：

[
\lambda^{US}*{JP,1} = \lambda^{US}*{JP,0} + \gamma_{US}\cdot(\tau_{JP}^{nom}-\tau_{JP,0}^{nom})
]

[
\lambda^{JP}*{JP,1} = 1 - \lambda^{US}*{JP,1} - \lambda^{MX}_{JP,1}
]

其中 (\gamma_{US}>0) 表示关税每上升 1 个百分点，日系向美国本土迁移的生产份额增加多少（由经验或情景假设给定）。

从美国角度看，日本品牌在美销售可以拆成：

* **进口车销量**：
  [
  Q^{imp}*{JP,1} = \lambda^{JP}*{JP,1},Q^{brand}_{JP,1}

  * \lambda^{MX}*{JP,1},Q^{brand}*{JP,1}
    ]
    （其中特别注意：来自墨西哥部分在关税上按墨西哥对美车关税算，通常享受 USMCA 优惠关税）
* **在美本土生产销量**：
  [
  Q^{USprod}*{JP,1} = \lambda^{US}*{JP,1},Q^{brand}_{JP,1}
  ]

这里 (Q^{brand}_{JP,1}) 可用“日本品牌总销量 = 日系进口 + 日系在美/墨生产”估算。如果数据实在不够细，可以把全部日系车简化成两个来源：“日系进口”和“日系本地化生产”。

**集成到主模型的方法：**

* 在 2.2 的来源国集合 r 中，把“日本车”细分成

  * “JP_import” （日本本土生产出口）
  * “JP_NA” （日系在北美生产的车）
* 对“JP_NA”部分关税按 USMCA / 零关税处理，对“JP_import”按互惠关税后日本税率处理。
* 日本通过提高 (\lambda^{US}*{JP}) 和 (\lambda^{MX}*{JP}) 来降低整体有效关税负担，从而减轻销量损失。

## 六、子模型 4：经济传导到美国汽车产业

这部分就是“**进口结构变化 → 国内产量/增加值/就业**”的传导。

### 4.1 进口渗透率与国产替代

设：

* 美国新车市场总支出：
  [
  E = \sum_r Q_{r} P_r
  ]
* 进口车支出：
  [
  M = \sum_{r\in \text{import}} Q_{r} P_r
  ]
* 进口渗透率：
  [
  \pi = \frac{M}{E}
  ]

关税上调后，计算新值 (\pi_1)，与基期 (\pi_0) 比较：

* 若总需求变化不大，**进口萎缩部分可视为由本土生产替代的潜在空间**：
  [
  \Delta Y_{\text{auto,US}} \approx (M_0 - M_1)\cdot \kappa
  ]
  (\kappa) 表示进口减少 1 美元有多少比例会转化为美国本土生产增加（可取 0.5–1 做情景分析）。

### 4.2 简化投入产出（IO）传导

可以用 **2–3 部门简化 IO 模型**：

* 部门 1：汽车制造（Auto）
* 部门 2：上游零部件与材料（Parts & Materials）
* 部门 3：其他行业（Rest）

假设给出 3×3 投入系数矩阵 (A)，终端需求向量 (y)，则总产出：

[
x = (I-A)^{-1} y
]

你的“冲击”是**美国汽车终端需求中本土生产部分增加**（相当于 (y_{Auto}) 增加、进口部分降低），把 (\Delta y_{Auto}) 带入就能得到：

* 汽车行业总产出增量 (\Delta x_{Auto})
* 上游零部件和材料产出增量 (\Delta x_{Parts})

就业可以用行业劳动系数 (e_{i})：

[
\Delta L_i = e_i \cdot \Delta x_i
]

这样就形成了从关税到**美国汽车行业产出与就业**的完整传导链条。

## 七、情景设计（模型怎么用）

至少设计三类情景：

1. **基准情景（无关税调整）**：按 2024 年实际税率和市场结构
2. **互惠关税 + 日本无应对**：日本完全将关税转嫁给美国消费者（(\theta_{JP}=0)，(\lambda^{US}_{JP}) 不变）
3. **互惠关税 + 日本中度应对**：部分吸收关税（(\theta_{JP}=0.5)），产能缓慢向美国和墨西哥转移（(\lambda^{US}_{JP}) 增加）
4. **互惠关税 + 日本强烈应对**：高比例吸收关税 + 大规模北美本地化生产

对每个情景，分别输出：

* 日本对美汽车出口量、出口额变化 (\Delta Q_{JP}, \Delta V_{JP})
* 美国进口结构中，日本、墨西哥、加拿大等占比变化
* 美国国内汽车产量、增加值与就业的变化（包括日系在美工厂 vs 美国本土品牌的份额变化）

这些指标就可以支撑问题二的分析和结论。

## 八、问题二所需数据及处理方法

这里重点说你项目里“官方提供的数据”怎么用，外加需要补充什么数据。

### 8.1 关税数据库（tariff_data_20xx.zip）

这些压缩包里是各年份的关税数据库 txt / xlsx，字段结构由 **《Tariff Database Code Key》** 和 **《USITC Tariff Database Fields》** 说明：

关键字段包括（只列与问题二相关的）：

* `HTS8`：8 位 HTS 税则号
* `BRIEF_DESCRIPTION`：简要商品描述
* `MFN_RATE_TYPE_CODE`：最惠国税率类型代码（对应 Code Key 中 A 部分的 0,1,2,3,4,5,6,7 等）
* `MFN_AD_VAL_RATE`：MFN 从价税部分（如 0.025 表示 2.5%）
* `MFN_SPECIFIC_RATE`、`MFN_OTHER_RATE`：从量税 / 其他税部分
* 各类优惠税率字段：如 `USMCA_*`（墨加）、`KOREA_*`、`JAPAN_INDICATOR`、`JAPAN_AD_VAL_RATE` 等，用于特定自贸协定
* `BEGIN_EFFECT_DATE`、`END_EFFECTIVE_DATE`：生效与失效时间，2024 版一般到 2050-12-31，2025 版会反映互惠关税变化

**处理步骤：**

1. **读入 2024 和 2025 年 txt / xlsx**

   * 2024：`trade_tariff_database_202405.txt`
   * 2025：`tariff_database_2025.txt`
     注意 2015–2016 是“|”分隔，2017 以后多为逗号分隔。

2. **过滤汽车相关税则号**
   根据 HTS 第 87 章文档，对应乘用车、商用车、零部件的 4/8 位码，例如：([Harmonized Tariff Schedule][2])

   * 8703：乘用车
   * 8704：货运汽车
   * 8708：零部件
     （可以视题目需要决定是否只做 8703 乘用车）

   在关税库中筛选 `HTS8` 前 4 位为上述编码。

3. **确定关税类型，计算有效从价率**

   * 对大部分汽车项目，`MFN_RATE_TYPE_CODE` 通常为 **7（单纯从价税）** 或 0（免税），即关税为：
     [
     \text{Duty} = \text{MFN_AD_VAL_RATE} \times \text{Import Value}
     ]
   * 如果出现 4、5、6 等“从价+从量”的组合，可有两种处理：

     * **简化**：只取从价部分 `MFN_AD_VAL_RATE`，认为从量部分占比很小
     * **精确**（若你有按数量的贸易数据）：
       依据 Code Key 给出的公式，例如类型 4：
       [
       \text{Duty} = \text{Specific rate} \cdot Q1 + \text{Ad Valorem rate} \cdot \text{Value}
       ]
       再用
       [
       \tau^{AVE} = \frac{\text{Duty}}{\text{Value}}
       ]
       计算**从价等价关税率**。

4. **区分 MFN 与日本 / USMCA 等优惠税率**

   * 对日本：使用 `JAPAN_AD_VAL_RATE`、`JAPAN_RATE_TYPE_CODE` 等字段，计算互惠关税前后的**日本专门税率**。
   * 对墨西哥、加拿大：使用 `USMCA_*` 字段。

5. **得到 2024 vs 2025 的加权平均关税**

   为每个来源国 r（至少包括 JP, MX, CA, KR, EU）在汽车商品集合 (i\in87xx) 上的贸易加权平均关税：

   [
   \tau_{r,t} = \frac{\sum_i \tau_{i,r,t} \cdot M_{i,r,t}}{\sum_i M_{i,r,t}}
   ]

   其中 (M_{i,r,t}) 是 i 类商品自 r 国进口到美的金额（来自 DataWeb 贸易数据，见下一小节）。
   实在没有按 8 位数据时，可以退而求其次：对 87 章整体用 2 位章级数据做加权，或用简单平均近似。

### 8.2 贸易数据（DataWeb‑Query‑Export/Import.xlsx）

你项目里两个 Excel 大致是：

* **DataWeb‑Query‑Export.xlsx**：`FAS Value` sheet：按 HTS 章节（比如 01, 02,...）和国家给出的出口额（2020–2025 年）
* **DataWeb‑Query‑Import.xlsx**：`General Import Charges` sheet：按 HTS 章节和国家给出的**关税收入**（General Import Charges，2020–2025 年）

从样例看，它们的结构类似：第一行是“Data Row Count”，第二行才是真正的“字段名称”（Data Type, HTS Number, Description, Country, 2020, ...）。你需要：

1. **重设表头**

   * 删除第 0 行（Data Row Count）
   * 把第 1 行作为新表头：

     * 列 1：Data Type
     * 列 2：HTS Number
     * 列 3：Description
     * 列 4：Country
     * 列 5–10：2020–2025 各年数值
   * 把“Total Exports|Annual Data”或“General Imports|Annual Data”这样的原列名替换为 `Data_Type`，方便后续处理。

2. **提取汽车相关数据**

   * 筛 `HTS Number = 87`（整章“车辆及其零件”）
   * `Country` 筛选：Japan、Mexico、Canada、Korea、Germany、EU countries 等。
   * 把“宽表”（列是年份）转成“长表”：每一行对应（年份 t，HTS Number 87，Country r，Value）
     这样你就能得到 2020–2025 年、按国家分组的**出口额**和**关税收入**。

3. **获得进口金额和数量数据**

   上述 Excel 主要是**出口值和关税收入**，对于问题二你更需要的是：

   * **美国汽车进口金额（Customs Value / CIF）**
   * **美国汽车进口数量（辆数/单位）**

   这部分可能需要你自己去 DataWeb 追加查询：选择

   * Trade Flow：`U.S. imports for consumption` 或 `General imports`
   * Data to report：`Customs Value`, `Quantity 1`（如果单位是辆）
   * Classification：HTS，Commodity list 选择 8703/8704 等精确税则号
   * Countries：Japan, Mexico, Canada, Korea, EU 等

   得到的结构与当前 Excel 类似，再做一次“重设表头 + 筛选 + 宽变长”的整理即可。([dataweb.usitc.gov][3])

   如果你不方便再查数据，也可以用题目给出的统计（比如 2024 年美国汽车进口额 2170 亿美元及各国份额）来做高层校准。([S&P Global][1])

4. **计算平均单价**

   有了金额 (V_{r,t}) 和数量 (Q_{r,t}^{units})，即可算出各国平均进口车价：

   [
   P_{r,0} = \frac{V_{r,2024}}{Q_{r,2024}^{units}}
   ]

   这就是前面 logit 模型里的 (P_{r,0})。

5. **验证关税率：用“关税收入 / 进口额”**

   如果你有同一维度的：

   * 进口关税收入：`General Import Charges` 数据表中的值
   * 进口金额：`Customs Value`

   可以进一步算出一个**实际有效关税率**（包含 MFN + 反倾销等一切加成）：

   [
   \tau^{eff}*{r,2024} = \frac{\text{Duties}*{87,r,2024}}{\text{ImportsValue}_{87,r,2024}}
   ]

   与关税数据库算出来的法定税率对比，既可以做合理性校验，也可以直接将其用于模型（避免处理复杂的具体税则结构）。

### 8.3 日本非关税应对所需的额外数据

为刻画日本非关税应对，你最好补充以下信息（可以来自公开报告、公司年报或新闻）：

* 日本主要车企在北美的**产能与产量**（丰田、本田、日产、马自达等在美国/墨西哥工厂的产量）
* 各品牌在美国销量中，**进口 vs 本土生产**的比例（类似“某品牌在美销售中 xx% 为进口”的统计）([S&P Global][1])
* 主要车企的毛利率和折扣策略，以支持你对“关税吸收比例 (\theta_{JP})”的情景设定

没有这些数据也没关系，你可以通过**合理假设 + 敏感性分析**（改变 (\theta_{JP})、(\lambda^{US}_{JP}) 的取值）来说明结论的稳健性。

### 8.4 构造最终用于建模的数据表

经过上述处理，你最终要得到一个“长表”，每一行对应某年 t、某来源 r（JP/MX/CA/KR/EU/US），内容包括：

* 年份 (t)
* 来源国 / 地区 (r)
* 商品类别（可选：8703 / 8704 / 87 整章）
* 进口数量 (Q_{r,t}^{units})
* 进口金额 (V_{r,t})
* 平均价格 (P_{r,t} = V_{r,t}/Q_{r,t})
* 法定关税率 (\tau^{stat}_{r,t})（来自 tariff_data 和 Code Key）
* 实际有效关税率 (\tau^{eff}_{r,t})（若有关税收入数据）
* 日本品牌在美本地生产份额 (\lambda^{US}_{JP,t})（情景或外部数据）

然后：

1. 用 2024 年数据标定 (s_{r,0}, P_{r,0}, A_r, \tau_{r,0})
2. 由 2025 年关税库计算关税调整后的 (\tau_{r,1})（互惠关税实施后）
3. 设定日本非关税应对参数 (\theta_{JP}, \lambda^{US}_{JP,1})
4. 代入前面 logit + 价格需求 + IO 模型，得到各情景下的美日贸易与美国汽车产业变化
